# Spotify Manual Testing Project
# Project Overview

        This project focuses on manual testing of the Spotify application.
        The goal is to identify bugs, verify functionalities, and ensure a smooth user experience.

# Features Tested
        User Login & Signup
        Search Functionality
        Play / Pause Songs
        Playlist Creation
        Song Controls (Next, Previous, Shuffle)
        Volume Control
        UI/UX Validation
# Testing Types
        Functional Testing
        UI Testing
        Usability Testing
        Regression Testing
# Bugs Identified

        Song lyrics have some spelling issues
# Project Structure
        Test Scenarios
        Test Cases
        Bug Reports
        Test Summary Report
# Tools Used
        Manual Testing
        MS Excel (Test Cases & Bug Reports)
        Browser (Chrome)
# Test Summary

        Testing was performed on key modules of Spotify.
        Bugs were identified and documented with proper steps and severity levels.

# Conclusion

        Spotify provides a good user experience, and works properly.
